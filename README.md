# Console App Assignment (Parts 1–6)

This is my C# Console Application for the Visual Studio course.  
The project includes all six parts of the assignment.

## What the program includes
- Part 1: Array, user input, and loops  
- Part 2: Infinite loop and fixed loop  
- Part 3: Loops using < and <=  
- Part 4: Search a list and show the first match  
- Part 5: Search a list with duplicates and show all matches  
- Part 6: Foreach loop that checks for unique and duplicate items  

## How to run
1. Open the project in Visual Studio  
2. Run the program (Ctrl + F5)  
3. Follow the prompts
